Inventory App

A simple PHP inventory/bin-tracking web app with image upload and auto-fill capabilities

Features

  • Bins CRUD - Each bin has a number and optional category label
  • Items CRUD - Title, description, vendor, price, product ID, URL, quantity, bin assignment
  • Photo Upload - Store item images under public/uploads/
  • Auto-fill (Beta) - Suggest bin based on keywords and fetch OpenGraph metadata from URLs
  • Pure PHP - PDO SQLSRV + simple HTML (Tailwind CDN), no framework required
  • Microsoft SQL Server - Works with SQL Server databases

Requirements

  • PHP 8.1+ with sqlsrv and/or pdo_sqlsrv extensions enabled
  • Microsoft SQL Server (local or remote)
  • A web server (Apache/Nginx) - Point your doc root to public/

Quick Start

  1. Database Setup
    • Create a database (e.g., inventory_app)
    • Run the schema: sql/schema.sql
    • Update DB credentials in app/config.php
  2. Deploy
    • Place this folder on your server
    • Set document root to public/
    • Visit http://localhost/
